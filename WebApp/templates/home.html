◾Este es mi proyecto web realizado con Python usando el framework de desarrollo web de Django.

◾También se mostrarán los distintos tipos de servicios haciendo como un ejemplo a una compañía de envíos.

◾Junto con el uso de apps extras y procesadores de contexto (context_processors.py) para el uso de variables globales.
